# 1.サービス名
BASE LOG（ベースログ）
野球の試合進行に係るタスク増加、人手不足を解決するサービス。
審判による判定（ボール、ストライク、アウト等）をリアルタイムで連携し、
スコアボードやアプリに同期したり、主審と塁審のコミュニケーションを円滑にすることを目指します。

## 3.内容
- ① テンキーをPCまたはスマホに接続し、主審用URLを開きます。テンキーで以下の数字を入力します。※テンキーのNumLockを有効にしてください
    - [8]：Ball（ボール）
    - [9]：Strike（ストライク）
    - [5]：Out（アウト）
    - [3]：Foul（ファール）
- ② ①で数字が入力されると、主審・塁審それぞれの画面に入力結果が表示されます。
- ③ 主審：
    - 「直前の操作を取り消す」ボタンを押すと、一つ前の状態に戻ります
    - 「打者出塁(カウント更新)」ボタンを押すと、アウトカウントはそのままでB(ボール)/S(ストライク)を0にします
    - ボタンを押すと、主審・塁審それぞれの画面にフィードバックが飛びます。
- ④ 塁審：
    - 「判定リクエスト」ボタンを押すと、主審・塁審それぞれの画面にフィードバックが飛びます。
- ⑤ 主審の入力したデータをマスタデータとする為、塁審はBSO判定へのリクエストのみ可能です。
- ⑥ 主審のデータ入力は、ピッチャーの球数をカウントすることを想定しています。ファールボールのカウントや、「打者出塁(カウント更新)」ボタンは、球数のカウントに活用します。
